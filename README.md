# 7thMac

7th Heaven wrapper for macOS

## Prerequisites

This wrapper at the moment only supports the [GOG release](https://www.gog.com/en/game/final_fantasy_vii).

## How to use

- Download the [latest release](https://github.com/julianxhokaxhiu/7thMac/releases/latest) and install the `7th Heaven.app` in your `Applications` folder
- Download the installer files from the [GOG release](https://www.gog.com/en/game/final_fantasy_vii)
- Launch `7th Heaven` and follow the prompts
- When asked to pick an installer pick the `setup_final_fantasy_vii_2.0_gog_v1_(88522).exe` file
- Accept the EULA and click on Install
- Click on Exit when the installer finishes
- When 7th Heaven launches, hit Save on the Settings window and wait for FFNx to be installed
- Go to `Settings -> Game Driver` and pick `Vulkan` as backend renderer
- Enjoy!
